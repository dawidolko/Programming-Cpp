### **Zadanie 2.**
Opisać działanie następującej funkcji:
```
double funkcjanowa(double a, double b, unsigned n)
{
  double zmienna1;
  if(b==1)
    return n*a;
  else
  {
    zmienna 1=a*(1=pow(b,n))/(1-b);
    return zmienna1;
  }
}
```

**Jaką wartość zwróci, gdy wywołamy ją z argumentami (2,3,4).**
